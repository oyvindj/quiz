Super simple Groovy and Springboot app that
demonstrates:
- Java and Groovy in same class structure
- Springboot
- WebService
- Thymeleaf UI
- Mongo DB
- Spring Mail w/ Google
- WebService Client
- Slf4j Logging
- Application Properties
- Automatic Reload w/ bootRun
- TODO SMS

BUILD
```
gradle build
```
RUN
```
gradle bootRun
```
